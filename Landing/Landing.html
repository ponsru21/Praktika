javaCub: Тест по языку java

javaCub - тест, разработанный для проверки знаний языка java. Тест содержит 5 вопросов и нацелен на новичков, только начинающих изучение языка. Тест можно пройти несколько раз, все результаты и дата выполнения теста будут записаны и сохранены.
Для сохранения своего результата введите имя (не менее 3-х символов).
